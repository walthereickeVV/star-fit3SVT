## 🌟 О ПРОЕКТЕ

**Star Fit** — это современный, стильный и полностью адаптированный сайт для фитнес-клуба в **Светлогорске**. 

**КЛЮЧЕВАЯ ОСОБЕННОСТЬ:** В клубе тренируют **Лукьяновы** — семья чемпионов мира по пауэрлифтингу и фитнесу.

| Тренер | Титулы | Специализация |
|--------|--------|---------------|
| 🏆 **Владимир Лукьянов** | МСМК, чемпион Мира, Европы, Евразии | Пауэрлифтинг, силовая подготовка |
| 🌸 **Яна Лукьянова** | Фитнес-трансформация, нутрициолог | Персональный фитнес, восстановление после родов |
| 🎵 **Татьяна Лукьянова** | Эксперт по технике, нутрициолог | Групповой фитнес, Zumba, Step, Stretching |

🔗 **Демо:** [https://star-fit.netlify.app](https://star-fit.netlify.app)  
📱 **Instagram:** [@starfit_svt](https://www.instagram.com/starfit_svt)
